# 学习路线
1. 阶段1
- 基础三大件: HTML CSS Javascript
- 计算机网络基础 TCP/IP协议 Http/Https协议
- 数据结构与算法
- 互联网的运行原理
- 浏览器的运作原理
2. 阶段2 
- CSS的预处理工具
  - 推荐Sass、PostCss.
- 前端的JS框架
  - 推荐React
  - 推荐Vue3.0(Vue4.0 过段时间可能会发布)
- CSS框架
  - 推荐 Antd
  - 推荐 Element UI
- CSS优化方案
  - Styled Component
  - CSS Modules
  - Styled JSX
3. 阶段3 
- 测试
  - 单元测试
  - 集成测试
  - E2E测试
- 代码质量
  - 类型校验 TypeScript
  - Flow
- Linter and Formater 
  - 推荐Eslint
- 编码规范
  - javascript Style Guide
  - CSS style Guidde
  - React Style Guide
- 性能
  - 后续更新
